よくある質問

Miyabiの導入に必要なものは何ですか？ ▼

GitHubアカウントとNode.js環境があれば、すぐに始められます。npx miyabi initを実行するだけで、必要な設定が全て自動で完了します。

既存プロジェクトにも導入できますか？ ▼

はい、miyabi installコマンドで既存プロジェクトに簡単に統合できます。既存のGitHub設定を自動検出し、最適な設定を提案します。

料金はかかりますか？ ▼

Miyabi自体は完全無料のオープンソースです。Claude APIを使用する場合は、Anthropic社の料金が別途かかります。

どのようなプロジェクトに適していますか？ ▼

TypeScript/JavaScript、Python、Go、Rustなど主要言語に対応。Next.js、React、Django、Flaskなどのフレームワークも自動検出します。

サポートはありますか？ ▼

GitHub Discussionsでコミュニティサポートを提供しています。詳細なドキュメントも充実しており、ほとんどの疑問はすぐに解決できます。
